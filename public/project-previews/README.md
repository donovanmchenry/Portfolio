# Project Preview Screenshots

Add screenshots of your projects here for the hover preview feature.

## Required Screenshots:

1. **njit-schedule-pro.png** - Screenshot of https://www.njitschedulepro.com/
2. **njit-empty-room.png** - Screenshot of https://njitemptyroomfinder.onrender.com/

## How to Add Screenshots:

1. Visit your project website
2. Take a full-page screenshot (or at least 1920x1080)
3. Save it with the filename specified above
4. Place it in this directory (`public/project-previews/`)
5. Refresh your portfolio to see the preview

## Tips:

- Use PNG or JPG format
- Recommended size: 1920x1080 or larger
- Make sure the screenshot shows the main features of your project
- You can use browser dev tools (F12) or screenshot tools
