# Order Up - Dynamic Card Matching Game

A memory-based web game that challenges players with exponentially increasing difficulty, featuring up to 12 ingredients per order with smooth animations and responsive design.

## Game Concept

**The Challenge:** Players must memorize and recreate complex food orders by matching ingredient cards in the correct sequence.

**The Twist:** Each level increases the number of ingredients exponentially, testing memory limits while maintaining engaging gameplay through smooth animations and immediate feedback.

### Animation System

Built a custom animation engine using CSS transforms and JavaScript timing:

```javascript
class AnimationController {
  constructor() {
    this.activeAnimations = new Set();
    this.animationQueue = [];
  }

  async revealCard(cardElement, delay = 0) {
    return new Promise(resolve => {
      setTimeout(() => {
        cardElement.classList.add('revealed');
        cardElement.addEventListener('transitionend', resolve, { once: true });
      }, delay);
    });
  }

  async sequentialReveal(cards, interval = 300) {
    for (let i = 0; i < cards.length; i++) {
      await this.revealCard(cards[i], i * interval);
    }
  }
}
```

### Difficulty Scaling Algorithm

Implemented progressive difficulty that maintains engagement:

```javascript
function calculateDifficulty(level) {
  const baseIngredients = 3;
  const maxIngredients = 12;
  
  // Exponential growth with ceiling
  const ingredients = Math.min(
    baseIngredients + Math.floor(level * 1.2),
    maxIngredients
  );
  
  // Adjust timing based on complexity
  const revealTime = Math.max(1000 - (level * 50), 400);
  const memoryTime = ingredients * 800;
  
  return { ingredients, revealTime, memoryTime };
}
```
